## 🚀 快速开始

### 启动应用

1. **确保后端服务运行**:
   ```bash
   cd /path/to/opencode
   python -m app.main
   ```

2. **打开浏览器**:
   访问 `http://localhost:8088`

3. **开始使用**:
   - 在输入框中输入你的任务描述
   - 点击"发送"按钮（或按 Enter）
   - 等待 AI 完成任务

## 🎯 功能概述

### 核心功能

- ✅ **AI 编程助手** - 自动生成代码、文件、项目

## 📖 使用指南

### 基本使用流程

#### 1. 创建新任务

1. 在输入框中输入任务描述，例如：
   ```
   创建一个 Python Flask 应用，包含用户登录功能
   ```

2. 点击"发送"按钮（或按 Enter）

3. AI 将开始执行任务：
   - 显示思考过程
   - 调用工具（read, write, bash 等）
   - 生成文件
   - 显示结果

#### 2. 追问和修改

AI 完成后，你可以继续提问：

```
数据库连接部分有问题，请使用 SQLite
```

AI 会基于之前的上下文进行修改。

#### 3. 查看生成的文件

- 在右侧文件列表中查看
- 点击文件名查看内容
- 支持语法高亮

### 高级功能

#### 文件预览（打字机效果）

当 AI 生成文件时：
- 自动弹出预览窗口
- 打字机效果展示代码生成
- 实时显示文件内容

**预览窗口功能**:
- 切换正常视图 / Diff 视图
- 启用/禁用语法高亮
- 启用/禁用缓冲优化
- 查看文件历史
- 在新窗口打开

#### 历史回溯

1. 点击"查看历史"按钮
2. 选择历史版本
3. 自动切换到 Diff 视图
4. 对比当前版本和历史版本

#### Diff 视图

- **绿色**: 新增的行
- **红色**: 删除的行
- **黄色**: 修改的行（显示删除线和新内容）

## ❓ 故障排除

### 常见问题

#### 1. AI 无响应

**可能原因**:
- 后端服务未启动
- 网络连接问题
- API 配置错误

**解决方法**:
1. 检查后端服务状态
2. 查看浏览器控制台错误
3. 检查网络连接

#### 2. 文件预览不显示

**可能原因**:
- 预览功能被禁用
- 文件类型不支持

**解决方法**:
1. 检查 `preview-config.js` 配置
2. 确认文件类型已启用预览

#### 3. 历史记录丢失

**可能原因**:
- 浏览器缓存被清理
- 使用了隐私模式

**解决方法**:
1. 使用正常浏览模式
2. 不要清除浏览器缓存

#### 4. SSE 连接断开

**可能原因**:
- 网络不稳定
- 后端服务重启

**解决方法**:
1. 检查网络连接
2. 刷新页面
3. 系统会自动重连

### 调试模式

打开浏览器控制台（F12）查看：
- **Network 标签**: 查看网络请求
- **Console 标签**: 查看日志和错误

## 🔌 API 文档

### 端点列表

#### Session 管理

```http
POST /opencode/session
GET /opencode/session/{id}
DELETE /opencode/session/{id}
GET /opencode/sessions
```

#### Message 管理

```http
GET /opencode/session/{id}/messages
POST /opencode/session/{id}/message
```

#### 事件流

```http
GET /opencode/events?session_id={id}
```

#### 历史回溯

```http
GET /opencode/get_file_history?session_id={id}&file_path={path}
GET /opencode/get_file_at_step?session_id={id}&file_path={path}&step_id={step}
```

#### 工具端点

```http
GET /opencode/health
GET /opencode/info
```

### 事件类型

| 事件类型 | 说明 |
|---------|------|
| `connection.established` | SSE 连接建立 |
| `message.updated` | 消息更新 |
| `message.part.updated` | 消息部分更新 |
| `preview_start` | 文件预览开始 |
| `preview_delta` | 文件预览增量 |
| `preview_end` | 文件预览结束 |
| `timeline_update` | 时间轴更新 |
| `ping` | 心跳事件 |
