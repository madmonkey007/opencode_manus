## 容器管理

### 查看日志

```bash
# 实时查看容器日志
docker logs -f opencode-container

# 查看应用日志
docker exec opencode-container tail -f /app/opencode/logs/app.log
```

### 停止/重启服务

```bash
# 停止容器
docker stop opencode-container

# 启动容器
docker start opencode-container

# 重启容器
docker restart opencode-container
```

### 使用 docker-compose

```bash
cd D:\Manus\opencode

# 停止服务
docker-compose down

# 启动服务
docker-compose up -d

# 查看日志
docker-compose logs -f
```
