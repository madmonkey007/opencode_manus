## 修复原理

**问题**: IIFE闭包内的函数暴露到全局作用域失败

**解决**: 创建独立的JS文件，直接在全局作用域定义函数

**history-fix.js**:
```javascript
window.loadSessionTimeline = async function(sessionId) {
    // 从API加载timeline
    // 调用processEvent渲染每个事件
};

window.handleHistorySessionClick = async function(sessionId) {
    // 加载messages
    // 调用loadSessionTimeline
};
```
