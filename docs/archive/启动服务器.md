## 手动启动服务器

由于Windows环境限制，请按以下步骤手动启动服务器：

### 方法1：使用批处理文件（推荐）

1. 双击运行：`D:\Manus\opencode\opencode_start.bat`
2. 等待窗口显示启动信息
3. 看到类似以下信息表示启动成功：
   ```
   INFO:     Started server process [12345]
   INFO:     Waiting for application startup.
   INFO:     Application startup complete.
   INFO:     Uvicorn running on http://0.0.0.0:8088
   ```

### 方法2：使用命令行

打开命令提示符（cmd.exe），执行：

```batch
cd D:\Manus\opencode
python -m uvicorn app.main:app --host 0.0.0.0 --port 8088 --log-level info
```

### 方法3：使用Python脚本

```batch
cd D:\Manus\opencode
python serve.py
```

## 启动后验证

1. 打开浏览器访问：http://localhost:8088
2. 按 F12 打开开发者工具，检查Console标签
3. 确认没有JavaScript语法错误
4. 测试创建新任务
5. 测试点击历史记录（不应该触发重复查询）

## 浏览器缓存清除

如果看到旧页面，请清除缓存：
- 按 `Ctrl+Shift+Delete`
- 选择"缓存的图片和文件"
- 点击"清除数据"

然后按 `Ctrl+F5` 强制刷新页面。

## 停止服务器

关闭服务器窗口，或按 `Ctrl+C`
