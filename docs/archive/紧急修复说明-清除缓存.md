## 立即执行以下步骤

### 步骤1: 清除浏览器缓存（必须！）

**Chrome/Edge浏览器**:
1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存的图片和文件"
3. 点击"清除数据"

**或者强制刷新**:
- 按 `Ctrl + Shift + R`（Chrome/Edge）
- 按 `Ctrl + F5`（Windows）

### 步骤2: 打开开发者工具禁用缓存

1. 按 `F12` 打开开发者工具
2. 切换到 `Network` 标签
3. 勾选 `Disable cache`
4. 刷新页面（`F5`）

### 步骤3: 访问应用

```
http://localhost:8089
```
