### **Phase 1: 提示词缓存**（0.5天）

**文件**: `app/prompt_enhancer.py`

**修改**:
```python
from functools import lru_cache
import hashlib

@lru_cache(maxsize=100)
def enhance_prompt_cached(prompt_hash: str, user_prompt: str, mode: str) -> str:
    """带缓存的提示词增强"""
    # 原有逻辑...
    pass

def enhance_prompt(user_prompt: str, mode: str = "auto") -> str:
    """提示词增强（带缓存）"""
    prompt_hash = hashlib.md5(f"{user_prompt}_{mode}".encode()).hexdigest()
    return enhance_prompt_cached(prompt_hash, user_prompt, mode)
```

**收益**: 0.3-0.5s（对重复提示词）
